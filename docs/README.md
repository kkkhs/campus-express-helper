## 💡 常见问题

### 我应该看哪个文档？

- **第一次部署**：先看 [../README.md](../README.md)，再看 [DEPLOY_README.md](./DEPLOY_README.md)
- **快速部署**：看 [QUICK_START.md](./QUICK_START.md)
- **从旧版升级**：看 [UPGRADE_GUIDE.md](./UPGRADE_GUIDE.md)
- **了解优化内容**：看 [OPTIMIZATION_SUMMARY.md](./OPTIMIZATION_SUMMARY.md)

